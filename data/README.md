# Данные

Сырые данные в репозитории не публикуются: датасет собран парсингом объявлений cian.ru и выгрузками из 2ГИС, распространение исходных файлов ограничено условиями использования этих сервисов. Ниже — схема, достаточная для воспроизведения пайплайна на собственных данных.

## all.csv — объявления о продаже квартир

7 604 строки, разделитель `;`. Омск, срез на момент сбора.

| Столбец | Тип | Описание |
|---|---|---|
| url | str | ссылка на объявление |
| price | int | цена, ₽ (целевая переменная) |
| total_meters | float | общая площадь, м² |
| living_meters | float | жилая площадь, м² (23 % пропусков) |
| kitchen_meters | float | площадь кухни, м² (8 % пропусков) |
| rooms_count | int | число комнат |
| floor | int | этаж |
| floors_count | int | этажность дома |
| year_of_construction | int | год постройки (15 % пропусков) |
| district | str | район города |
| street | str | улица |
| house_number | str | номер дома |

Пропуски в исходнике закодированы как `-1` или пустая строка. Столбцы с долей пропусков выше 30 % (object_type, heating_type, house_material_type, finish_type, residential_complex) удаляются на этапе очистки.

## coordinates.csv — геокодированные адреса

| Столбец | Тип | Описание |
|---|---|---|
| address | str | адрес вида «Омская область, Омск, {улица}, {дом}» |
| Latitude | float | широта |
| Longitude | float | долгота |

Внимание: join с объявлениями идёт по строковому совпадению адреса, поэтому адрес формируется без `strip()` — ровно в том виде, в каком геокодировался.

## 2гис_{категория}.csv — точки интереса (POI)

Четыре файла: `2гис_еда.csv`, `2гис_здоровье.csv`, `2гис_культура.csv`, `2гис_образование.csv`. Итого 1 235 объектов после фильтрации по рубрикам и границам Омска.

| Столбец | Тип | Описание |
|---|---|---|
| Наименование | str | название организации |
| Рубрики | str | рубрики 2ГИС через разделитель |
| Широта | float | координата |
| Долгота | float | координата |

Используемые рубрики: кафе, рестораны, больницы, медицинские центры, кинотеатры, музеи, театры, школы, гимназии, лицеи, детские сады, университеты, колледжи.
